- Whisper is a speech recognition tool based on [[AI]] and created by  [[OpenAI]]
-
- ### Prerequisites
	- [[Python]]
	- [[ffmpeg]]
-
- ### Installation
	- To install Whisper, execute the following:
		- pip install whisper
-
- ### Usage
	- ### Transcribe file
		- ```
		  whisper twc-088.mp3 --model medium --language English > twc-088.srt
		  ```
